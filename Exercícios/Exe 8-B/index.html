Código-fonte / pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Como tal, usando a tag code, podemos tornar o comando mono-espaçado.

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

No exemplo acima, apesar de usarmos a tag code, o HTML não lê o código escrito com indentação, quebras de linha etc.
Para resolvermos isso, envelopamos o código com a tag PRE, e veja o resultado: 

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')
