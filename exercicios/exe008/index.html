Principais formatações

Negrito / destaque

Há duas maneiras: uma que carrega conteúdo semântico — usando a tag STRONG e outra cujo uso é apenas um recurso visual, estilístico — usando a tag B

Itálico / ênfase

Temos o uso de um termo em itálico usando a tag I (não semântico)

Nesta frase temos um termo com ênfase, carregando um valor semântico com ele, usando a tag EM.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.
E no outro parágrafo, temos outro texto marcado com style.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Antigamente existia a tag <strike>, que hoje também está depreciada pela última versão da HTML. No lugar dela entrou a tag <DEL>, que significa que o texto está ali, pode até ser lido, mas deve ser desconsiderado pelo leitor. Exatamente como eu acabei de fazer.

Texto inserido

Nesse caso, se colocarmos um texto qualquer dentro de <ins>, estamos dizendo que o texto está ali, deve ser lido e você deve prestar atenção nele. É o que fazíamos anteriormente ao utilizar a tag <u> para simplesmente sublinhar um texto.

Texto sobrescrito

Para inserir coisas do tipo x2+3 — tag SUP

Texto subscrito

Para inserir coisas do tipo H2O — tag SUB

Dica ExXxtra

Lembre-se, caso aconteça de escrever um texto e quiser adicionar uma tag posteriormente, basta selecioná-lo e abrir a paleta de comandos (ctrl+shift+P), selecionar o emmet "wrap with abbreviation" e escolher a tag desejada! O nome desse processo é "envelopar"
